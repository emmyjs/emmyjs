# Adding Emmy.js to your Ruby on Rails project <i class="devicon-rails-plain colored"></i>
          
## Quick Start
1. Install the `emmy-dom` npm package:
```bash
npm i emmy-dom
```
2. Use importmap to import the `emmy-dom` package:
```bash
./bin/importmap pin emmy-dom --download
```
